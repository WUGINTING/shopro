# Shopro 前端 Vue.js 應用程式

這是 Shopro 電商管理系統的前端應用程式，使用 Vue 3、TypeScript、Pinia 和 Vite 構建。

## 功能模組

### 1. CRM 模組 (會員管理)
- **會員管理**: 新增、編輯、刪除、查詢會員
- **會員等級管理**: 管理會員等級與權益
- **積點系統**: 積點發放、兌換、調整
- **EDM 電子報**: 建立和發送電子報
- **部落格管理**: 管理部落格文章

**API 路徑**: `/api/crm/*`

### 2. Product 模組 (商品管理)
- **商品管理**: 商品 CRUD 操作
- **商品分類**: 支援多層級分類結構
- **商品標籤**: 標籤管理與商品標籤關聯
- **批次操作**: 批次更新商品狀態、價格等

**API 路徑**: `/api/products/*`, `/api/product-categories/*`, `/api/product-tags/*`

### 3. Order 模組 (訂單管理)
- **訂單管理**: 訂單 CRUD 操作
- **訂單查詢**: 依狀態、客戶、日期範圍查詢
- **訂單統計**: 訂單數據分析與統計
- **批次操作**: 批次更新訂單狀態、批次刪除

**API 路徑**: `/api/orders/*`

## 安裝與執行

### 前置需求
- Node.js 18+ 或 20+
- npm 或 yarn

### 安裝依賴
```bash
npm install
```

### 開發模式運行
```bash
npm run dev
```

應用程式將在 `http://localhost:5173` 啟動。

### 構建生產版本
```bash
npm run build
```

### Lint 檢查
```bash
npm run lint
```

## 環境配置

編輯 `.env` 檔案來配置後端 API 基礎 URL：

```env
VITE_API_BASE_URL=http://localhost:8080
```

## 💡 開發 Tips

### 使用 Quasar 組件
```vue
<!-- 表格 -->
<q-table :rows="data" :columns="columns" selection="multiple" />

<!-- 對話框 -->
<q-dialog v-model="show" persistent>
  <q-card>...</q-card>
</q-dialog>

<!-- 通知 -->
$q.notify({ message: 'Success', type: 'positive' })

<!-- 確認 -->
$q.dialog({
  title: '確認刪除?',
  message: 'Are you sure?',
  ok: true,
  cancel: true
})
```

### 常見 API 模式
```typescript
// 創建 API 模組
export const productApi = {
  async getAll() { ... },
  async create(data) { ... },
  async update(id, data) { ... },
  async delete(id) { ... },
  async batchImport(file) { ... },
  async export(format) { ... }
}
```

### 表單驗證
```vue
<q-form ref="form" @submit="handleSubmit">
  <q-input
    v-model="form.name"
    label="名稱"
    :rules="[val => !!val || '必填']"
  />
</q-form>
```
